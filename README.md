## Build

Place this directory at `device/google/stallion`
inside a current TWRP source tree, then run:

```sh
source build/envsetup.sh
lunch twrp_stallion-eng
mka vendorbootimage
```

For GKI devices this builds recovery into `vendor_boot`. Check your TWRP source
branch if it uses a differently named vendor-boot target.
Generation is a starting point, not proof that touch, encryption, display,
kernel modules, AVB signing, or flashing are correct for the physical device.
